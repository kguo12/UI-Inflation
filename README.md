# Inflation's Impact on the Unemployment Insurance Program
Data and analysis repository

[Definitions by DOLETA](https://oui.doleta.gov/unemploy/data_summary/definitions.pdf)

`master.csv` contains relevant data for analyses. The master.csv Update notebook contains functions to update master.csv with additional UI data from subsequent years, as well as calculate real values for any reference year (choose to be the most recent year in the sample: 2024).
